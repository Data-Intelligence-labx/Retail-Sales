# Retail Sales Analysis 

## Overview

This project focuses on analyzing retail sales data to uncover actionable insights around revenue performance, product behavior, customer purchasing patterns. The objective is to transform raw transactional data into meaningful business intelligence that supports data-driven decision-making.

The analysis emphasizes clean data modeling, reproducible analytics, and business-oriented metrics, following industry best practices used in production BI environments.

## Objectives

1. Analyze overall sales performance and revenue trends

2. Identify top- and bottom-performing products

3. Evaluate customer purchasing behavior

4. Detect invalid or anomalous transactions

5. Generate KPIs relevant to retail decision-making

## Key Insights Covered

1. Sales trends over time (daily / monthly)

2. Product-level performance analysis

3. evenue contribution and ranking

4. Order value and quantity analysis

5. Data quality validation (negative sales, cancellations, etc.)

## Tools & Technologies

- Power BI - Data modeling, DAX measures, and visualization

- Excel - Data extraction and preprocessing

- DAX - KPI calculations and advanced filtering logic



## Metrics & KPIs

1) Total Sales / Net Sales

2) Average Order Value (AOV)

3) Top N Products by Revenue

4) Quantity Sold

5) Invalid Transaction Flags
